# vm.applescript

Simple script to launch five terminal tabs and ssh into a username@host given via the commandline

## Usage

`./vm.applescript brian@example.org`

## Tip

Set `alias launch_vm="~/vm/vm.applescript user@host"` in your `.bash_profile` (or equivalent) for extra laziness.
